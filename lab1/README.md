## Формулировка задания

Реализовать дерево с возможностью:
- Добавления узлов к заданному родителю
- Удаления узлов из дерева (вместе с поддеревом)
- Вывода дерева на экран
- Подсчёта числа листьев дерева (вариант 24)

Лист — узел, не имеющий дочерних узлов. Программа подсчитывает общее количество листьев в дереве.

## Описание выполненной работы

### Структура проекта

Проект состоит из следующих файлов:

#### 1. `main.c` - Основная программа и реализация дерева

Содержит структуру `Node`, все функции для работы с деревом и интерактивное меню.

Структура узла:
```c
typedef struct Node {
    int key;
    struct Node* parent;
    struct Node* firstChild;
    struct Node* nextSibling;
} Node;
```

**Основные функции:**
- `create_node()` — создание нового узла
- `find_node()` — поиск узла по ключу (обход в глубину)
- `insert_node()` — добавление узла к заданному родителю
- `print_tree_recursive()` — вывод дерева с отступами в виде ASCII-дерева
- `free_subtree()` — освобождение памяти поддерева
- `delete_by_key()` — удаление узла вместе с поддеревом
- `count_leaves()` — подсчёт числа листьев (вариант 11)

Реализация функции задачи:
```c
static int count_leaves(Node* root) {
    if (!root) return 0;

    if (root->firstChild == NULL) {
        return 1;
    }

    int sum = 0;
    Node* child = root->firstChild;
    while (child) {
        sum += count_leaves(child);
        child = child->nextSibling;
    }
    return sum;
}
```

#### 2. `tests.c` - Тесты

Тесты для проверки функций дерева. Подключает `main.c` через `#include` с переопределением `main`.

Тест-кейсы:
- `test_create_root` — создание корня дерева
- `test_insert_child` — добавление дочернего узла
- `test_reject_duplicate_key` — отказ при дублирующемся ключе
- `test_delete_subtree` — удаление узла с поддеревом
- `test_count_leaves` — подсчёт числа листьев

#### 3. `Makefile` - Сборка проекта
